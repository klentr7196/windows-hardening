# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this tool, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email: **security@obsidiancorps.com**

We will acknowledge your report within 48 hours and provide a fix timeline.

## Scope

This tool modifies Windows registry settings, services, scheduled tasks, and Defender policies. While all changes are reversible (via the built-in backup/restore), incorrect modifications could affect system behavior.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | Yes       |
| < 1.0   | No        |
